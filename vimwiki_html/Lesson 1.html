Hello World

Hello world is a ubiqitous first program programmers often write to test out a new idea or when trying a new language.

The idea of hello world is actually quite useful and the idea is to simplify a problem as much as possible so you can have a starting point that works. For example if you have some large complicated code, you might do some small isolated hello-world type test where you rip out only the problematic code and wrench on that.

Since you are embarking on a code learning adventure it's only proper to begin with a hello world program.

For this course we'll use Python since its popular and used widely

Code

hello-world.py 

print("Hello world!");

output 

Hello world!
